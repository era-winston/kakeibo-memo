プライバシーポリシー

家計メモ帳（Kakeibo Memo）

1. はじめに

本プライバシーポリシーは、家計メモ帳（以下「本アプリ」）における ユーザーの個人情報の取り扱いについて説明するものです。 本アプリは、ユーザーのプライバシーを最大限に尊重し、 個人情報の保護に努めています。

2. データの収集について

本アプリは、ユーザーの個人情報を一切収集しません。

  • アナリティクスツール・広告SDK等の第三者サービスは使用していません
  • ユーザーの利用状況や行動を追跡・記録することはありません
  • 外部サーバーへのデータ送信は一切行いません

3. データの保存

本アプリで入力された取引データ、予算設定、テンプレートなどの情報は すべてユーザーの端末内にのみ保存されます。

  • クラウドサーバーやインターネットを介したデータ同期は行いません
  • データはアプリのローカルストレージ（Hive データベースおよび SharedPreferences）に保存されます
  • 他のアプリからこれらのデータにアクセスすることはできません

4. データのエクスポート

本アプリにはCSVエクスポート機能があります。 この機能はユーザーが明示的に操作した場合にのみ実行され、 端末のOS標準の共有機能を通じてファイルが出力されます。 エクスポート先はユーザー自身の選択に委ねられます。

5. セキュリティ機能

本アプリには以下のセキュリティ機能があります：

  • パスコードロック：4桁のPINコードによるアプリロック機能。PINはSHA-256ハッシュ化して端末内に保存され、平文では保存しません。
  • 生体認証：端末のFace ID / Touch ID / 指紋認証を利用したロック解除。生体情報は端末のOS側で管理され、本アプリが直接取得・保存することはありません。

6. データの削除

ユーザーはいつでもアプリ内の設定画面から「すべてのデータを削除」を実行することで、 保存されたすべてのデータを完全に消去できます。 また、端末からアプリをアンインストールすることで、 すべてのデータが自動的に削除されます。

7. 子供のプライバシー

本アプリは特定の年齢層を対象としたものではありませんが、 個人情報を一切収集しないため、児童のプライバシーに関するリスクはありません。

8. ポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。 重要な変更がある場合は、アプリのアップデートを通じてお知らせします。
